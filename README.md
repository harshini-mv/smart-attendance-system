# Smart Attendance System

A modern Smart Attendance System built using React, TypeScript, and Vite.

## Live Demo
smart-attendance-system-82z6jacze-manniharshini-7720s-projects.vercel.app

## Features

- Student Login & Signup
- Attendance Tracking
- Biometric Verification
- Geolocation-based Attendance
- Analytics Dashboard
- Responsive UI

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- React Router DOM

## Installation

```bash
npm install
npm run dev
```
